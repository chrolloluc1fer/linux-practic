# Linux Commands — DevOps Core (с пояснениями)

> Цель: понимать систему и уметь диагностировать проблемы.

---

## 1. Навигация и файлы

- `pwd` — текущая директория  
- `ls` / `ls -la` — список файлов (включая скрытые)  
- `tree` — древовидный вывод структуры  
- `cd <dir>` — перейти в директорию  
- `mkdir <dir>` — создать директорию  
- `rm <file>` / `rm -r <dir>` — удалить файл / директорию  
- `cp <src> <dst>` / `-r` — копирование  
- `mv <src> <dst>` — перемещение / переименование  
- `touch <file>` — создать пустой файл  
- `file <file>` — тип файла  
- `stat <file>` — подробная информация (inode, права, время)

---

## 2. Чтение файлов

- `cat` — вывести файл целиком  
- `less` — постраничный просмотр  
- `head -n 10` — первые строки  
- `tail -n 10` — последние строки  
- `tail -f <file>` — следить за логом в реальном времени  
- `nl` — показать строки с номерами  

---

## 3. Потоки и перенаправления

- `>` — перезаписать вывод в файл  
- `>>` — добавить в файл  
- `2>` — записать ошибки в файл  
- `&>` — stdout + stderr в файл  
- `|` — передать вывод одной команды другой  
- `tee` — вывести и записать одновременно  
- `xargs` — передать список аргументов другой команде  

---

## 4. Поиск

- `grep "text"` — поиск текста  
- `grep -r` — рекурсивный поиск  
- `find /path -name "*.log"` — поиск файлов  
- `which <cmd>` — где находится бинарник  
- `whereis <cmd>` — путь + man  

---

## 5. Архивы

- `tar -czf archive.tar.gz dir/` — создать архив  
- `tar -xzf archive.tar.gz` — распаковать  
- `gzip` / `gunzip` — сжатие файла  
- `zip` / `unzip` — zip-архив  

---

## 6. Пакеты (Ubuntu)

- `apt update` — обновить список пакетов  
- `apt upgrade` — обновить систему  
- `apt install <pkg>` — установить  
- `apt remove <pkg>` — удалить  
- `apt search <pkg>` — поиск  
- `dpkg -l` — список установленных  

---

## 7. Пользователи и права

- `useradd` / `userdel` — создать / удалить пользователя  
- `usermod` — изменить пользователя  
- `passwd` — сменить пароль  
- `groupadd` — создать группу  
- `id <user>` — UID, GID  
- `groups <user>` — группы пользователя  
- `chmod 755 file` — изменить права  
- `chown user:group file` — изменить владельца  
- `umask` — маска прав по умолчанию  
- `su - user` — войти как пользователь  
- `sudo <cmd>` — выполнить от root  

---

## 8. Процессы

- `ps aux` — список процессов  
- `top` / `htop` — мониторинг в реальном времени  
- `kill <PID>` — SIGTERM  
- `kill -9 <PID>` — SIGKILL  
- `pkill <name>` — убить по имени  
- `bg` / `fg` — фон / вернуть в foreground  
- `jobs` — фоновые процессы  
- `nice` / `renice` — приоритет процесса  

---

## 9. Ресурсы и система

- `uptime` — время работы + load average  
- `free -h` — память  
- `df -h` — дисковое пространство  
- `df -i` — inode  
- `du -sh *` — размер директорий  
- `iostat` — I/O статистика  
- `vmstat` — память + процессы  
- `lscpu` — CPU  
- `lsblk` — блочные устройства  

---

## 10. Сеть

- `ip a` — IP адреса  
- `ip route` — маршруты  
- `ping` — проверка связи  
- `traceroute` — маршрут до хоста  
- `ss -tlnp` — кто слушает порт  
- `netstat -tlnp` — аналог (старый)  
- `curl` — HTTP-запрос  
- `wget` — скачать файл  
- `nslookup` / `dig` — DNS  

---

## 11. SSH

- `ssh user@host` — подключиться  
- `ssh-keygen` — создать ключ  
- `ssh-copy-id` — добавить ключ на сервер  
- `scp` — копирование по SSH  
- `rsync` — синхронизация файлов  

---

## 12. Systemd

- `systemctl start/stop/restart <service>`  
- `systemctl status <service>`  
- `systemctl enable/disable <service>`  
- `journalctl -u <service>` — логи сервиса  
- `journalctl -xe` — последние ошибки  

---

## 13. Логи

- `/var/log/` — директория логов  
- `journalctl` — логи systemd  
- `dmesg` — сообщения ядра  
- `logrotate` — ротация логов  

---

## 14. Диски и ФС

- `mount` / `umount` — монтирование  
- `/etc/fstab` — автоподключение  
- `lsblk` — устройства  
- `blkid` — UUID  
- `fdisk` — разметка диска  
- `tune2fs` — параметры ext4  

---

## 15. Bash полезное

- `history` — история команд  
- `alias` — псевдонимы  
- `env` / `printenv` — переменные окружения  
- `export` — установить переменную  
- `watch` — повторять команду  
- `crontab -e` — планировщик задач
